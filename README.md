# Шахматы:

## Цель проекта

Создать десктопное приложение, которое позволяет записывать и проверять ходы, хранить историю игр, вычислять рейтинг игроков и статистику.

---

## Игровое поле

- Размер: 8x8
- Координаты по горизонтали: a - h
- Координаты по вертикали: 1 - 8

---

## Фигуры

| Фигура | Количество |
|--------|------------|
| Король | 1 |
| Ферзь | 1 |
| Ладья | 2 |
| Слон | 2 |
| Конь | 2 |
| Пешка | 8 |

---

## Функциональные требования

1. Проверка корректности ходов по правилам шахмат
2. Запись всех ходов партии
3. Хранение истории игр в базе данных (SQLite)
4. Рейтинг игроков (система Эло)
5. Статистика по игрокам: победы, поражения, ничьи, процент побед
6. Графический интерфейс: доска 8x8, ввод ходов мышью
