# Шахматы:

## Цель проекта

Создать десктопное приложение, которое позволяет записывать и проверять ходы, хранить историю игр, вычислять рейтинг игроков и статистику.

---

## Игровое поле

- Размер: 8x8
- Координаты по горизонтали: a - h
- Координаты по вертикали: 1 - 8

---

## Варианты игры (точки расширения)

Перед началом партии игрок может выбрать один из вариантов правил:

| Вариант | Особенности |
|---------|-------------|
| Классика (FIDE) | Стандартные правила, рокировка, взятие на проходе |
| Шахматы-960 (Фишера) | Случайная начальная расстановка, особые условия рокировки |

---

## Функциональные требования

1. Проверка корректности ходов по правилам шахмат
2. Запись всех ходов партии
3. Хранение истории игр
4. Рейтинг игроков
5. Статистика по игрокам: победы, поражения, ничьи, процент побед
